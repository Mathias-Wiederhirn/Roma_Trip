# Tania, Eleni & Mat's Rome Trip

![Rome trip preview](https://commons.wikimedia.org/wiki/Special:FilePath/Saint%20Peter%27s%20Square%2C%20Vatican%20City%20-%20April%202007.jpg?width=1200)

A small travel website for our Rome stay, with an interactive map, day-by-day agenda, hotel details, gluten-free restaurant ideas, and family-friendly notes.

## What's inside

- `index.html`  
  Main website with the interactive trip planner

- `rome_trip_itinerary_print.html`  
  Working copy of the itinerary

## Website features

- Interactive map centered on the hotel
- One-day-at-a-time itinerary view
- Hover details for important stops
- Google Maps links for visits, meeting points, and restaurants
- Gluten-free restaurant suggestions
- Family safety notes for daytime and evening
- Mobile-friendly layout

## Open the site

Open [`index.html`](./index.html) in your browser.

## Publish with GitHub Pages

1. Push the repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Save.

Your site will then be published as a simple travel website you can open on desktop or mobile.
